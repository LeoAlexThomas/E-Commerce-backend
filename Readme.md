# E-Commerce Backend [User Auth]

## Description

This project is a backend application for managing e-commerce users data. It allows users to register, login, get User info through a RESTful API.

## Features

- Authentication and Authorization
- Database integration for persistent storage
- Input validation and error handling

## Technologies Used

- Node.js
- Express.js
- MongoDB (or your chosen database)
- Mongoose (for MongoDB object modeling)
- JWT (jsonwebtoken)
- bcrypt
- Postman (for API testing and documentation)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/LeoAlexThomas/E-Commerce-backend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd E-Commerce-backend
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Set up your database configuration in `Database/dbConfig.js`.
5. Start the application:
   ```bash
   npm start
   ```
